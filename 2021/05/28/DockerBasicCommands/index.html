Docker Basic Commands

Docker安装测试

docker run debian echo "hello world!"
  • 本地没有debian镜像，先pull
  • 由debian镜像创建容器
  • 执行命令行echo "hello world!"

基本指令

docker images

查看本地镜像

docker images

docker search

搜索合适的镜像

docker search ubuntu

docker pull

拉取镜像源

docker pull ubuntu

docker run

根据镜像构建容器

docker run --name myDebian -h Debian -it debian /bin/bash
参数 含义
-h 指定容器的hostname
-i 以交互模式运行容器，通常与 -t 同时使用
-t 为容器重新分配一个伪输入终端，通常与 -i 同时使用
/bin/bash docker后台必须运行一个进程，否则容器就会退出，所以启动容器后启动bash

docker ps

列出容器(默认正在运行的)

docker ps
参数 含义
-a 显示所有容器（包括未运行的）
-l 显示最近创建的容器
-n + int 列出最近创建的n个容器

docker top

查看容器中运行的进程信息

docker top myDebian

docker logs

获取容器日志

docker logs --since="2021-05-25" --tail=10 myDebian
参数 含义
—since 显示某个开始时间的所有日志
-t 显示时间戳
—tail 仅列出最新N条容器日志

动态跟踪容器日志

docker logs -tf --tail=10 myDebian
参数 含义
-f 跟踪日志输出

docker inspect debian

docker stop

关闭容器

docker stop myDebian

docker start

启动容器

docker start myDebian

docker exec

在运行的容器中开启一个交互终端

docker exec -it myDebian /bin/bash

在运行的容器中执行命令或脚本

docker exec -it myDebian echo "hello world!!!"

Ctrl + P + Q

退出交互式终端但是保证容器在后台继续运行

docker attach

进入正在运行的容器

docker attach myDebian

docker rm

删除单个容器

docker rm -f myDebian

删除所有已停止的容器

docker rm $(docker ps -a -q)

docker rmi

删除单个镜像

docker rmi -f debian
